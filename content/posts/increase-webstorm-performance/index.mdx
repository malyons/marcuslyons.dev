title: Increase WebStorm Performance
excerpt: Garbage collection might go on strike.
date: 2021-03-30
published: true
keywords: [webstorm, garbage collection, optimization]
---

Apply the following settings in WebStorm config file.

```shell
-Xmx2048m
-Xss256k
-XX:+UseG1GC
-XX:InitiatingHeapOccupancyPercent=65
-XX:G1HeapRegionSize=2
-XX:MaxGCPauseMillis=100
```
